Quick start

Post with a few
lines of code

One API call. Every platform. Node.js, Python, Go, PHP, cURL — pick your language.

ayrshare-quickstart.js 
// npm install @ayrshare/social-post-api

const SocialPost = require('@ayrshare/social-post-api');
const api = new SocialPost('YOUR_API_KEY');

const response = await api.post({
  post: "Today is a great day!",
  platforms: ["twitter", "facebook", "instagram", "linkedin"],
  scheduleDate: new Date().toISOString(),
});

console.log(response.id);       // "post_abc123"
console.log(response.postIds);  // { twitter: "...", facebook: "..." }
# pip install ayrshare

api = SocialPost("YOUR_API_KEY")

    "post": "Today is a great day!",
    "platforms": ["twitter", "facebook", "instagram"],
    "scheduleDate": "2026-02-19T12:00:00Z",
})

print(response["id"])       # post_abc123
print(response["postIds"])  # {"twitter": "...", "facebook": "..."}

// go get github.com/ayrshare/ayrshare-go

package main

import (
  "fmt"
  ayrshare "github.com/ayrshare/ayrshare-go"
)

func main() {
  client := ayrshare.NewClient("YOUR_API_KEY")
  resp, _ := client.Post(ayrshare.PostRequest{
    Post:      "Today is a great day!",
    Platforms: []string{"twitter", "facebook"},
  })
  fmt.Println(resp.ID) // post_abc123
}
API capabilities

Everything exposed
as REST endpoints

No SDK required. If you can make an HTTP request, you can use Ayrshare.

01 POST/post Publish text, images, videos, or links to 13+ platforms simultaneously. Schedule for future delivery or post immediately.
02 DELETE/post Remove posts across all networks with one API call. Supports bulk deletion by date range or post ID.
03 GET/analytics Full history and engagement stats for every post — impressions, clicks, shares, and reach across all connected platforms. New
04 POST/comment Retrieve, add, and manage comments on any post programmatically. Moderate at scale without logging in.
05 POST/message Send direct messages and platform DMs. Build inbox management and automated response workflows.
06 POST/webhooks Real-time event notifications when posts publish, fail, or receive engagement. Build reactive pipelines.
07 GET/auto-hashtags AI-powered hashtag suggestions based on trending data and your content. Maximize organic reach automatically.
08 POST/ads-boost Convert posts into Facebook Ads and Google Business promotions programmatically. Pro
REST surface

Clean, predictable endpoints

Standard REST conventions. JSON in, JSON out. No custom query languages or proprietary formats.

POST/api/post

Create and publish a post to one or more platforms

GET/api/post

Retrieve post history, status, and platform-specific IDs

DELETE/api/post

Delete a post from all platforms by Ayrshare post ID

GET/api/analytics/post

Fetch engagement metrics for any post across all platforms

POST/api/comment

Add or reply to comments on published posts

GET/api/comment

Retrieve all comments for a given post or profile

GET/api/user

Fetch connected profile details, follower counts, bios

POST/api/shorten

Shorten URLs with click tracking before including in posts
