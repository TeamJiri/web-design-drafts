Quick start

Post with a few
lines of code

One API call. Every platform. Node.js, Python, Go, PHP, cURL — pick your language.

// npm install @ayrshare/social-post-api

const SocialPost = require('@ayrshare/social-post-api');
const api = new SocialPost('YOUR_API_KEY');

const response = await api.post({
  post: "Today is a great day!",
  platforms: ["twitter", "facebook", "instagram", "linkedin"],
  scheduleDate: new Date().toISOString(),
});

console.log(response.id);       // "post_abc123"
console.log(response.postIds);  // { twitter: "...", facebook: "..." }
# pip install ayrshare

print(response["postIds"])  # {"twitter": "...", "facebook": "..."}

// go get github.com/ayrshare/ayrshare-go

package main

import (
  "fmt"
  ayrshare "github.com/ayrshare/ayrshare-go"
)

func main() {
  client := ayrshare.NewClient("YOUR_API_KEY")
  resp, _ := client.Post(ayrshare.PostRequest{
    Post:      "Today is a great day!",
    Platforms: []string{"twitter", "facebook"},
  })
  fmt.Println(resp.ID) // post_abc123
}

API capabilities

Everything exposed
as REST endpoints

No SDK required. If you can make an HTTP request, you can use Ayrshare.

  • 01POST /postPublish text, images, videos, or links to 13+ platforms simultaneously. Schedule for future delivery or post immediately.
  • 02DELETE /postRemove posts across all networks with one API call. Supports bulk deletion by date range or post ID.
  • 03GET /analytics newFull history and engagement stats for every post — impressions, clicks, shares, and reach across all connected platforms.
  • 04POST /commentRetrieve, add, and manage comments on any post programmatically. Moderate at scale without logging in.
  • 05POST /messageSend direct messages and platform DMs. Build inbox management and automated response workflows.
  • 06POST /webhooksReal-time event notifications when posts publish, fail, or receive engagement. Build reactive pipelines.
  • 07GET /auto-hashtagsAI-powered hashtag suggestions based on trending data and your content. Maximize organic reach automatically.
  • 08POST /ads-boost proConvert posts into Facebook Ads and Google Business promotions programmatically.
